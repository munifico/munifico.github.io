### 고정 소수점
소수점 뒤 자릴수를 지정하는 형식을 원한다면 toFixed()를 사용합니다.

```javascript
x.toFixed(3);   // 19.510
x.toFixed(2);   // 19.51
x.toFixed(1);   // 19.5
x.toFixed(0);   // 20
```

이 숫자는 버림<sup>truncation</sup>이 아니라 반올림<sup>round
</sup>입니다.
